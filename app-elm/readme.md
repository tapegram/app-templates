# app-elm

## Dev Flow

### Running the app

You can use Elm's built in `elm reactor` though you have to manually refresh.

Alternatively, you can install and run `elm-live`:

```
npm i -g elm-live
```

Then run it like
```
elm-live <elm-file>
```

```
elm-live src/Main.elm
```

See: https://www.elm-live.com/
